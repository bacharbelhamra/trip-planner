# Trip Planner

Application web intelligente pour générer des itinéraires de voyage avec IA.

## Technologies
- React
- Firebase
- Groq API
- Unsplash API
- Google Maps

## Installation
npm install
npm run dev

## Fonctionnalités
- Génération automatique de voyages
- Carte interactive
- Sauvegarde des voyages
- Authentification utilisateur
